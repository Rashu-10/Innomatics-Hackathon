# Innomatics-Hackathon
This repository contains my solution for the Innomatics Research Labs Hackathon. It includes data loading from CSV, JSON, and SQL files, data merging using Pandas, and analysis on the final food delivery dataset.
## Steps Performed
1. Loaded orders data from CSV file.
2. Loaded user data from JSON file.
3. Loaded restaurant data from SQL file.
4. Merged all datasets using appropriate keys with left joins.
5. Created a final consolidated food delivery dataset.
6. Performed data analysis to answer business and analytical questions.
## Tools Used
Python, Pandas, SQLite
